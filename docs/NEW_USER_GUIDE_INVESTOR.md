---

### 🗣️ 第 2 句：策略靠不靠谱

> **"ETF 验证一下"** 或 **"这个策略打分多少？"**

**AI 收到后会做什么**：
- 跑 4 个验证器（Walk Forward / Monte Carlo / Cross ETF / Consistency）
- 给你一个综合分（0~1）

**输出长这样**：

```
综合分：0.426（未通过）
Walk Forward：0.33  ← 时序验证
Monte Carlo：0.80   ← 路径稳健
Cross ETF：0.33    ← 跨 ETF 泛化
Consistency：0.56   ← 时段一致
```

**你怎么用**：分 < 0.6 = 策略不及格，**不要用它做实盘**。这是 SKILL 帮你避坑。

---

### 🗣️ 第 3 句：现在怎么操作

> **"我的持仓要不要卖？"** 或 **"再平衡建议"**

**AI 会查你的持仓**（512170 等），告诉你：
- **止盈**：赚够了，要不要落袋为安
- **止损**：亏到止损线，要不要割肉
- **持有**：还没到触发条件，继续拿
- **清仓**：超过持仓天数上限，必须卖

**输出长这样**：

```
持仓 1 只：
  512170：HOLD（继续持有）
  - 止损：未触发
  - 止盈：未触发
```

**你怎么用**：每天看一下，决定今天要不要交易。

## 五、关键概念速查（4 个就够）

### 概念 1：C21-1 策略

**就是**："布林带中轨 + 60 日均线"买入规则
**作用**：判断"这只 ETF 现在该不该买"
**限制**：没有卖出信号（要靠止损/止盈）

### 概念 2：综合分（0~1）

**就是**：4 个验证器的加权平均
**门槛**：0.6 = 及格线
**用法**：分越高越值得信任

### 概念 3：4 验证器

| 名字 | 问的问题 |
|------|---------|
| Walk Forward | 时序上**未来还能用吗**？ |
| Monte Carlo | **行情剧烈波动**还能赚钱吗？ |
| Cross ETF | **别的 ETF**也能用吗？ |
| Consistency | **任何时段**都稳定吗？ |

### 概念 4：市场模式

- `trend_up`：上涨——可以追
- `trend_down`：下跌——**该减仓**
- `range_bound`：震荡——**高抛低吸**
- `crash`：暴跌——**立刻清仓**

## 七、FAQ（你一定会问的）

**Q1：AI 给 BUY，我直接买吗？**

A：❌ 不要。AI 给你**信号**，你**自己判断**。可以加一句"为什么这只 BUY"，让 AI 解释。

**Q2：综合分 0.426，能赚钱吗？**

A：⚠️ 不能。综合分 < 0.6 表示策略**未通过验证**——这是 SKILL 帮你避免踩坑的信号。

**Q3：数据过期 3 天，还能信吗？**

A：⚠️ 不能。数据过期时信号不可靠。让 AI "刷新数据"或第二天 14:30 后再用。

**Q4：我不懂代码，能学量化吗？**

A：✅ 能。这个 SKILL 就是为不懂代码的人设计的。你只需要**问 + 看 + 决定**。

**Q5：我想自己加个策略，怎么做？**

A：跟 AI 说 "我想加一个'股价突破 20 日新高'的策略"，AI 帮你写代码 + 测试 + 接入。
