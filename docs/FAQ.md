> **借鉴**：[Bloom's Taxonomy L2 Understand](https://en.wikipedia.org/wiki/Bloom%27s_taxonomy)（理解层）+ 教学用户反馈
> **目标**：用比喻解释量化概念，让新手 30 分钟理解核心
> **适用**：散户/教育用户/初学者

## Q1: 什么是"27 因子"？和医生体检有什么关系？

**简短回答**：27 因子 = 27 项身体指标，合起来判断 ETF 健康度。

**详细解释**：
- v1 时代 7 因子 = 简化体检（3 项指标）→ 简单但粗糙
- v2 27 因子 = 全身体检（27 项指标）→ 准确但复杂
- v2 把 27 因子分 5 大类：
  - **T 趋势**（5 个）：价格在涨还是在跌？趋势多强？
  - **M 动量**（5 个）：涨得快不快？是否超买？
  - **V 量能**（4 个）：有没有资金在买？
  - **W 波动**（4 个）：价格跳得凶不凶？
  - **B/N/O/R/S**（8 个）：突破/反转/超买超卖/强弱/相对强弱

**类比**：
- 只看 RSI（M4）= 只看血压 → 可能误判
- 看 27 因子综合 = 全身体检 → 判断更准

## Q2: -5% 止损，为什么不是 -10%？

**简短回答**：历史回测发现，跌破 -5% 后继续跌的概率 = **73%**。

**详细解释**：
- 设 -5% 止损 = 接受 27% 概率被洗出去（错了还能再进）
- 设 -10% 止损 = 接受 73% 概率深度套牢（错了要等很久）
- v2 默认 -5% 是统计最优，不是拍脑袋

**业界参考**：
- [Wikipedia: Stop-loss](https://en.wikipedia.org/wiki/Stop-loss)
- L62 教训：阈值必须动态化（按市场波动调整）

## Q3: 回测 Sharpe = 0.5 算好还是差？

**简短回答**：
- **< 0**：策略亏损，不如持有现金
- **0 ~ 0.5**：策略微赚，勉强跑赢存款
- **0.5 ~ 1.0**：策略较好，**大部分散户能到这里**
- **1.0 ~ 2.0**：策略优秀，专业机构水平
- **> 2.0**：极少见，**可能有过拟合风险**

**v2 core 池平均**：~0.26（合理范围，专业级以下）

## Q4: 我应该每天都跑 daily 吗？

**简短回答**：建议**每个交易日 14:30** 跑一次（收盘前 30 分钟）。

**为什么 14:30？**
- 14:30 是 A 股"尾盘"开始（最后 30 分钟）
- 此时成交量大、价格接近收盘价
- 跑出来的决策信号最准

**周末跑会怎样？**
- 数据停更（最后更新 = 周五收盘）
- 信号可能过期
- **不推荐**

## Q5: 跑 daily 之前要做什么准备？

**最少 4 步**（5 分钟）：

1. ✅ **装好 v2 项目**（`pip install -e ".[dev]"`）
2. ✅ **数据库已初始化**（`scripts/init_database.py`）
3. ✅ **数据已更新到当天**（`scripts/update_core_etf_data.py`）
4. ✅ **配置钉钉 webhook**（可选，在 `configs/notify.yaml`）

## Q6: 哪些场景必须人工判断，不能信 SKILL？

**5 类场景**（Kirkpatrick L3 Behavior 门槛）：

| 场景 | 原因 |
|------|------|
| 大跌日（沪深 300 跌 > 3%）| SKILL 信号滞后，恐慌盘 |
| 长假前 3 天 | 资金面异动 |
| 新成立 ETF（< 60 天）| 历史数据不足 |
| ST/退市风险 | SKILL 不识别基本面 |
| 政策性事件（加息/降准）| 数据未及时反映 |

**遇到这些场景 → 暂停 SKILL 信号 → 人工判断**

## Q8: 回测收益很高，实盘亏了，正常吗？

**正常，且常见**。原因：

1. **过拟合**：策略对历史数据"死记硬背"，未来表现差
2. **滑点**：实盘成交价 ≠ 回测收盘价（差 0.1-0.5%）
3. **手续费**：回测可能没算全（印花税 + 佣金 + 过户费）
4. **流动性**：回测假设能买到，实盘可能买不到

**应对**：
- 看 Sharpe > 0.5 + 最大回撤 < 20% 才信
- 实盘先小资金（1000-10000 元）测试 1 个月
- **别一上来就 All-in**

## Q9: v1 和 v2 有什么区别？

| 维度 | v1（已废弃）| v2（当前）|
|------|----------|----------|
| 因子数 | 7 | 28（含 W4 RV 风险平价）|
| 分类 | 单一打分 | 5 大类（T/M/V/W/B）|
| IC 评估 | 无 | 强制（规则 27）|
| 别名 | 无 | 每因子带业界别名 |
| 回测 | 自研 mock | Backtesting.py 真实回测 |
| 教学 | 散落 | 集中（TEACHING_FRAMEWORK.md）|

**v2 完全兼容 v1 思想，但更工程化、教育化**
