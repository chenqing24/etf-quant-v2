# utils — 工具

> **职责**：通用工具 + 安全审计
> **依据**：v1 utils/ 7 文件 + L101/L118-L120 教训

## 核心组件

| 组件 | 角色 |
|------|------|
| `db.py` | SQLite 封装 |
| `http.py` | HTTP 封装（限流 + 重试）|
| `logger.py` | 日志 |
| `safety_gate.py` | 写入前校验（机制层强制）|
| `audit_logger.py` | 审计日志 |
| `execution_source.py` | --source 标识（L101 防串扰）|

## 标准注释规范

每个 .py 文件头部 6 段：
1. 用途
2. 被谁调用
3. 功能说明
4. 使用方式
5. 依赖
6. 注意事项

## 关联教训

- L101（多执行源无标识 = 串扰）
- L118-L120（文档一致性 + 隐私保护）
